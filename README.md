# Dreame_Vacuum_Box
Reusable vacuum box for the Dreame L10S Ultra.  
Door gasket 1.75mm TPU filament.  
Hinge pin 1.75mm solid filament.  
Gasket TPU filament.  
Filters Dreame D9, L10 Pro.  

Note: THIS DESIGN IS CURRENTLY UNTESTED
